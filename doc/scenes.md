Сцена является основной конструктивной единицей игры.

Именно на сцене происходит отрисовка экрана, создание объектов, логика их взаимодействия и управление со стороны пользователя.

# флаги

actived - показывает, что сцена активна, без этого сцена будет исключена из хука update
created - показывает, что сцена создана, без этого сцена будет исключена из цикла
mounted - показывает, что сцена еще не была смонтирована, нужен для вызова первого запуска сцены в хуке mount
removed - показывает, что сцена была помечена на удаление и будет удалена в конце хука update

# жизненный цикл

для сцен у нас есть стек

- create

выполняется один раз при добавлении сцены в стек

- remove

выполняется один раз при удалении сцены из стека

затем запускается бесконечный игровой цикл game loop

в этом цикле происходит управление сценами

- mount

событие mount сцены выполняется один раз, уже в игровом цикле, когда сцена была добавлена в стек

для каждого этапа жизненного цикла game loop выполняются события:

- update
- render

эти события вызывают соответствующие события в каждой сцене, начиная сверху стека.

Например, если стек содержит сцены:

- OptionsScene
- GameScene

события выполняются в следующем порядке:

- OptionsScene.update
- GameScene.update
- OptionsScene.render
- GameScene.render

добавление сцены в стек происходит командой

scenes.add(GameScene)

удаление новой сцены из вершины стека происходит командой

scenes.remove()

также можно удалить сцену по ее названию командой

scenes.removeByName(name)

очистить стек можно командой

scenes.removeAll(name)
